Project: veselinka

Goal:
Create a party game for Android TV.

Players connect using their phones.

TV screen:
- shows QR code
- shows room code
- shows questions

Phones:
- join room
- receive tasks
- send answers

Players: 1–8

Game modes:
- questions
- truth or dare
- voting
- speed mini games

Stack:
Node.js
Socket.io
HTML
Simple design

Target:
Low hardware devices
Android TV WebView app later
