# Game Flow

1 Start game on TV

2 TV shows:
- QR code
- room code

3 Players join using phone

4 Players enter nickname

5 Host starts game

Game loop:

Round start

Server chooses game mode:
- question
- truth or dare
- vote
- speed game

TV shows question

Players receive task on phone

Players answer or vote

Server collects answers

TV shows results

Next round
